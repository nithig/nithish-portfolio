# **App Name**: Harsh's Orbiting Portfolio

## Core Features:

- Hero Section Grid: Renders a grid pattern background with a sea-blue spotlight effect on hover.
- Fixed Navigation Bar: Implements a fixed top navigation bar with a logo, center-aligned links, and a right-aligned resume CTA button.
- Animated Headline: Displays the hero headline with a blinking cursor and typewriter effect.
- Orbiting Tech Icons: Animates tech icons around a central image using rotating dashed circular borders.
- Floating Action Button: Creates a fixed floating action button at the bottom-right with pop-up social media icons, and a tooltip on hover.

## Style Guidelines:

- Primary color: Bright blue (#3B82F6) for interactive elements and highlights, providing a tech-forward feel.
- Background color: Deep dark blue (#0B1229), creating a modern, high-contrast backdrop for content. 
- Accent color: Cyan (#06B6D4) for the spotlight effect and subtle accents, contrasting against the primary blue. 
- Body font: 'Inter' sans-serif, suitable for both headlines and body text
- Lucide-react icons for the FAB and pop-up icons.
- Responsive design using Tailwind CSS to stack vertically on mobile and side-by-side on desktop.
- Use Framer Motion for smooth icon rotations, pop-up animations, and button effects.
